# @travelhub/shared

Pure, framework-free utilities usable by both `apps/web` and `apps/api`.
Never contains React, Express, or any I/O (database, HTTP, filesystem)
— those belong in their respective apps.

**Sprint 1 status:** scaffolded, empty. No business logic has been added
per this sprint's explicit scope (see `SPRINT_1` notes in the root
`README.md`).
